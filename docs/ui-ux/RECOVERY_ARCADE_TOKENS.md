## Shape

- Buttons radius: 16dp
- Cards radius: 24dp
- Pills (chips): 999dp

## Spacing

- Screen horizontal padding: 16dp
- Card padding: 16dp
- Vertical rhythm: 12dp / 16dp

## Typography

- Screen title: 24sp–28sp, bold
- Section title: 12sp–14sp, medium/bold, optional all-caps
- Metric value: 24sp–32sp, bold
- Labels/helper: 12sp–14sp, muted
